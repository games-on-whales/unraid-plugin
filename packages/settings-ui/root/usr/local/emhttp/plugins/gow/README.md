# Games on Whales — Unraid plugin

Stream games from your Unraid server to [Moonlight](https://moonlight-stream.org) clients. Deploys **Wolf** and **Wolf Den** via Docker Compose with a settings page under **Settings → Games on Whales**.

## What you get

- GPU picker and one-click **Install**
- ROM / BIOS / library path setup with Unraid folder browse
- Dashboard: container status, pairing, health check, Start / Stop / Update
- **Advanced**: Fix mounts, cleanup sessions, repair ES-DE, reconfigure, reset
- Optional Wolf memory caps (OOM protection)

## Requirements

- Unraid 6.12+
- [Compose Manager](https://forums.unraid.net/topic/114415-plugin-compose-manager/) plugin
- Intel, AMD, or NVIDIA GPU ([NVIDIA FAQ](https://github.com/games-on-whales/unraid-plugin/blob/main/docs/FAQ.md))
